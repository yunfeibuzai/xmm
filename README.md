​🚀 麦卖云 V2bX 后端全能王牌脚本
​这是一款专为 V2bX/v2bx 后端运维设计的“全能型”管理脚本。它不仅是一个安装器，更是一个涵盖了资源防爆控制、网络优化、深度日志净化、高并发并发调优的一站式生产力工具。

​✨ 为什么选择这款脚本？
​全系统自适应：原生支持 Systemd (Ubuntu/Debian/CentOS) 与 OpenRC (Alpine) 双模式。
​内核级防爆 (Anti-Explode)：独创资源护卫策略，通过 GOMEMLIMIT 限制内存，CPUQuota 压制性能，防止节点爆内存导致服务器宕机。
​深度日志净化：集成了模块化日志处理引擎，配合 logrotate 每小时强控，永不爆盘。
​极致网络优化：一键集成豪华版 BBR 加速、TCP 高并发连接数永久解锁。
​全透明运维：拒绝“盲人运维”，日志全链路透明化，错误排查一目了然。
​模块化架构：核心脚本轻量，动态调用外部高级配置模块，升级无需重装。

​📥 极速安装
​在 Root 权限下，执行以下一行命令即可：

一键安装👇🏻

wget -N https://raw.githubusercontent.com/yunfeibuzai/xmm/main/maimai && bash maimai

bash <(curl -fsSL https://raw.githubusercontent.com/yunfeibuzai/xmm/main/maimai)


安装完成后，输入 maimai 或 v2bx 即可随时呼出管理菜单。
